# Resell — UI Design Spec

**Date:** 2026-07-13
**Scope:** Full web UI — all pages in `apps/web`

---

## Design System

### Typography
- **Font:** Outfit (Google Fonts) — weights 400, 500, 600, 700, 800, 900
- **Import:** `https://fonts.googleapis.com/css2?family=Outfit:wght@400;500;600;700;800;900&display=swap`
- **Scale:** 4.5rem hero headline · 1.6rem page titles · 0.875rem body · 0.72rem meta · 0.6rem labels/badges
- **Letter-spacing:** -0.04em to -0.055em on large headings; +0.08–0.12em on uppercase labels

### Color Tokens
```
background:       #070707
surface:          #0d0d0d
surface-hover:    #111111
border:           #181818
border-subtle:    #111111

text-primary:     #e8e8e8
text-secondary:   #888888
text-muted:       #555555
text-disabled:    #3a3a3a

accent:           #10b981   (emerald)
accent-hover:     #0ea472
accent-bg:        #10b98108
accent-border:    #10b98130
accent-glow:      #10b981aa

danger:           #f87171
danger-bg:        #f8717108
danger-border:    #f8717130

warning:          #f59e0b
warning-bg:       #f59e0b08
warning-border:   #f59e0b30
```

### Spacing
- Page horizontal padding: `2rem`
- Section vertical gap: `1.25rem`
- Row padding: `0.75rem 2rem`
- Card padding: `1.25rem 1.5rem`

### Borders & Radius
- All borders: `1px solid #181818`
- Cards/inputs: `border-radius: 8–10px`
- Badges: `border-radius: 4px`
- Pills/chips: `border-radius: 999px`
- Buttons: `border-radius: 6–8px`

### Shadows & Effects
- Logo dot glow: `box-shadow: 0 0 8px #10b981`
- Hero dot-grid: `radial-gradient(circle, #1d1d1d 1px, transparent 1px)` at 26px×26px
- Hero fade: `linear-gradient(to bottom, transparent 50%, #070707 100%)` overlay
- Search focus ring: `border-color: #10b98150`
- Row hover: 2px left border `#10b981`, `background: #0d0d0d`

## Pages

- Active: emerald text + bottom border

**"In demand" section** (3-column featured grid):
- Section label: `IN DEMAND` — `0.6rem` uppercase, `#4a4a4a`, with `<hr>` line
- Each card: gradient thumbnail (dark green/amber/purple per category) + emoji, badges (category + "Selling fast"/"New"), name, venue+date, price + listing count
- `background: #070707`, hover `#0c0c0c`

**All events table**:
- Section label: `ALL EVENTS · {CITY}` with `<hr>`
- Column headers: `0.58rem` uppercase `#3a3a3a`

- Date: big day number `#d0d0d0` + small month `#555`
- Event name: `#aaa` → `#f0f0f0` on hover
- Meta row: category badge + venue name in `#555`
- Price: emerald, bold
- Count: `#4a4a4a` — "3 left" or "2 left" in `#f8717180` (scarce)

## Components

- Selling fast: red tint
- New: emerald tint
- Active: emerald
- Pending: amber

### Cards (surface)
- `background #0d0d0d`, `border 1px solid #181818`, `border-radius 8–10px`

## Implementation Notes

- Use `next/font/google` for Outfit (zero CLS)
- Set `<html class="dark">` — dark mode only, no toggle
- Define all tokens as CSS custom properties in `globals.css`
- Tailwind config: extend with the above color tokens
- No external component library — all components hand-coded with Tailwind utility classes
- shadcn/ui only for complex components if needed (date picker, select)
